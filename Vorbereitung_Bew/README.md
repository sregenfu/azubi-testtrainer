# Vorbereitung_Bew

Ausgelagertes Mini-Projekt fuer mehrere Ausbildungsarten.

## Struktur
- `index.html` -> Hub-Startseite fuer alle Ausbildungsarten
- `dialogmarketing-allgemeine.html` -> Allgemeine Infos + Testsimulationen
- `bewerbung-dialogmarketing.html` -> Bewerbungsseite
- `interviews.html` -> Interviewtraining mit Filtern und Mini-Karten
- `interviews-print.html` -> Druckansicht fuer Interviewkarten
- `formelsammlung.html` -> Mini-Karteikarten fuer Prozentrechnung, Dreisatz und Digital-Marketing-KPIs
- `sprachfokus-alternativen.html` -> Alternativen bei starker Sprachbegabung und schwaecherer Mathe
- `app.webmanifest` / `sw.js` / `icons/` -> PWA-Dateien fuer Tablet-Nutzung

## Nutzung
1. `index.html` im Browser oeffnen.
2. Ausbildungsart auswaehlen (aktuell Dialogmarketing).
3. In der Testseite mit Namen anmelden und Modus waehlen.
4. Interviewtraining, Formelsammlung und Bewerbungsseite direkt ueber die obere Navigation aufrufen.

Hinweis: Lernfortschritt wird lokal im Browser gespeichert (localStorage).
